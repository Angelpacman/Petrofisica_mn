el programa litodensidad.py es un programa para graficar M vs N en 
un plano de 2 dimensiones siendo N el eje x y siendo M el eje y.

el programa litodensidad_3D.py pretende graficar 3 ejes M, N y L.
siendo el el eje Z

para correr el programa es necesario instalar python junto con las 
librerias de: numpy, matplotlib, pandas, shapely, scipy.
estas librerias se instalan de manera automatica si se instala python 
desde la pagina de anaconda: https://www.anaconda.com/distribution/#download-section
adicionalmente desde anaconda se debe instalar las librerias: "shapely" y "scipy"


existe otra forma de instalar python numerico y para ello es necesario bajar python 3.6
de su pagina oficial. 
https://www.python.org/downloads/

Una vez descargado e instalado python se instalan 
las librerias de python necesarias:
numpy, matplotlib, pandas, shapely, scipy

si la variable de entorno de python est√° declarada (instalada) de manera global
se puede abrir una terminal y escribir los sigientes comandos para 
poder instalar los requerimientos:

pip install numpy
pip install pandas
pip install matplotlib
pip install Shapely                (cuidado, Shapely va con S mayuscula en algunos casos)
pip install scipy


CONSIDERACIONES:
este programa recibe como variables de entrada a un archivo .csv de nombre: eval_petro.csv

tener precisamente estos nombres:

PROF	LLD	LLS	FR	DT	NPHI	RHOB	GR

el archivo csv debe de estar en la misma carpeta junto con el programa litodensidad.py
